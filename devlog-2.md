# Devlog #2 — Building the Apps
**Date:** June 13, 2026

## Apps Built (10 total!)

### Terminal
A fully functional terminal emulator with:
- Command history (arrow up/down)
- 15+ commands: help, echo, date, ls, cat, neofetch, calc, cowsay, color, history, clear, whoami, uname, exit
- Colored output (prompt, success, error, info)
- Auto-scroll and click-to-focus

### Calculator
- Standard calculator layout (4x5 grid)
- Expression display + result display
- Supports: +, -, *, /, %, decimals
- Clear (C) and backspace (CE)
- Safe expression evaluation (no eval injection)

### Notepad (Text Editor)
- Full textarea with monospace font
- Toolbar: New, Save (downloads .txt), Bold, Uppercase, Lowercase, Word Count
- Line/column position indicator in status bar
- Auto-saves to localStorage between sessions

### File Manager
- Virtual filesystem with nested directories
- File type icons (documents, images, music, video, config)
- Navigation: back button, home button, path display
- Double-click folders to navigate in
- Click to select files, status bar shows item count

### Browser
- Built-in "Meridian Browser" with URL bar
- Internal pages: home, about, news, games, chat, music, settings
- Quick-link grid on homepage
- Navigation buttons (back, forward, refresh)
- Custom meridian:// protocol

### Weather
- Current conditions display (temp, description, location)
- Detail cards: humidity, wind, UV index
- 5-day forecast with icons

### Settings
- Tabbed sidebar: Appearance, Display, Sound, About
- Working accent color picker (changes system-wide!)
- Toggle switches for dark mode, desktop icons, sounds
- Wallpaper cycling (5 gradient themes)
- System info display

### Music Player
- Album art with rotation animation
- Track info, progress bar, time display
- Play/pause, previous, next controls
- Clickable playlist (5 tracks)
- Visual state: art spins when playing

### Paint
- HTML5 Canvas drawing
- 10 color palette with active indicator
- Brush size slider (1-20px)
- Brush and Eraser tools
- Clear canvas and Save as PNG
- Dark canvas background

### System Popups
- Volume slider popup
- Battery status popup
- Network toggle
